# Claim: Cross-Room Invariant Signal

**Status:** 🟡 Pending
**Source:** friendly-fox / plato-experience / PheromoneTrail.find_invariant()
**Priority:** Medium

---

## The Proposition

> *If the same path works in multiple rooms for the same desire, it's real structure (not coincidence).*

Cross-room invariants are the fleet's highest-confidence discovery signal. When two agents in different rooms independently discover the same path for the same desire, the path has survived a natural form of replication — it's not room-specific.

---

## Falsification Criteria

Falsified if:
1. Cross-room invariants have the same false-positive rate as single-room paths.
2. Cross-room replicability inflates confidence in false signals (the "echo chamber" problem).
3. The invariance signal doesn't correlate with ground truth when known-false paths are planted.

---

## Methodology

Plant known-false paths across multiple rooms. Measure:
- Do cross-room invariants ever preserve false signals?
- What's the false-positive rate for 2-room invariants vs. 3-room vs. 5-room?
- At what room count does cross-room invariance become reliable?

---

## Jester's Challenge

> *"What if cross-room invariance cements groupthink? The same wrong answer in five rooms is still wrong — it just has more social proof."*

The jester is right: cross-room invariance tests consensus, not truth. The test must include "planted false" paths that span rooms to see if invariance inflates false confidence.
